# Nour Ahmed Ibrahim — Animated Portfolio

Animated React portfolio for a Junior .NET Backend Developer.

## Built With

- React
- Vite
- Framer Motion
- Lucide React
- Custom CSS animations

## Main Sections

- Hero
- About
- Skills
- Featured Project: 7wshly
- Other Projects
- Experience, including Freelance .NET work and ITI training
- Contact

## How to Run Locally

```bash
npm install
npm run dev
```

Then open the local Vite URL in your browser.

## How to Build

```bash
npm run build
```

## How to Deploy on Vercel

1. Create a new GitHub repo, for example: `nour-portfolio`.
2. Upload these files to the repo.
3. Go to Vercel.
4. Import the GitHub repo.
5. Framework preset: Vite.
6. Build command: `npm run build`.
7. Output directory: `dist`.
8. Deploy.

## What to Update Before Applying

- Replace placeholder demo links with real demo video links.
- Add real 7wshly GitHub repo link if it becomes public.
- Add screenshots or screen recordings if available. If you have the 7wshly presentation, export real app screens from it and replace the visual placeholders.
